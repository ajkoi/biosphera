# Biosphera

Un jeu sur le theme de l'évolution et de l'anatomie comparée.

Fait dans le cadre d'un projet en enseignement scientifique en terminal.


## Commandes
### Déplacements
Z, Q, S et D pour se déplacer

### Interactions
#### Les cours
A pour interagir avec les maisons pour avoir des cours

#### Les cartes
E pour ouvrir le menu de l'arbre
R pour ouvrir le menu des cartes


# Comment jouer ?

Téléchargez le fichier correspondant à votre systeme d'expoitation dans Releases, et lancez le.
